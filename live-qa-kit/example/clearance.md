## 부록 — 복붙 요청 문구 전문

### CL-01

```
안녕하세요. TeamBoard stage 환경 라이브 QA를 준비 중입니다.
SSO 테스트 계정 2개가 필요합니다 — 관리자 역할 1개, 일반 멤버 역할 1개.
용도는 권한 경계 검증이며, test-board 보드 안에서 qa- 접두 테스트 데이터만 생성·정리합니다.
07-10(금)까지 회신 부탁드립니다. 계정 정보는 메일·채팅이 아닌 사내 시크릿 공유 절차로 전달해 주세요.
```

### CL-02

```
안녕하세요. TeamBoard stage 라이브 QA용으로 스토리지 API 키 1개가 필요합니다.
스코프는 테스트 버킷(tb-qa-bucket) 한정, 객체 생성·조회·삭제 권한 포함이어야 합니다
(업로드 파일의 실물 존재 교차검증과 테스트 종료 시 잔여 0건 정리검증에 사용).
07-10(금)까지 회신 부탁드립니다. 키는 사내 시크릿 공유 절차로 전달해 주세요.
```

### CL-03

```
안녕하세요. TeamBoard stage(https://stage.teamboard.example) 접근을 위해
QA 작업 PC의 내부망(VPN) 접근 개통을 요청드립니다. 기간은 QA 수행 기간(약 4주)입니다.
07-08(수)까지 개통 가능 여부 회신 부탁드립니다.
```

### CL-04

```
안녕하세요. TeamBoard stage 라이브 QA에서 검증 증거(DB 실제 반영 확인) 수집을 위해

07-10(금)까지 회신 부탁드립니다. 접속 정보는 사내 시크릿 공유 절차로 전달해 주세요.

```

