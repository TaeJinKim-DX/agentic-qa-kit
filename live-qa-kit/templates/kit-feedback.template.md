# kit-feedback.md — 킷 개선 제안 (이 프로젝트에서 발견)

> 킷 자체(live-qa-kit/)는 수정 금지(AGENT.md §0). 대신 킷을 쓰다 발견한 **일반화 가능한**
> 개선점을 여기 모은다. 사람이 검토해 킷 원본(별도 repo)에 반영한다. 이 프로젝트에만 해당하는
> 커스터마이징은 여기 말고 qa-config.yaml에 남긴다.
>
> 판단 기준: "다른 프로젝트에서도 똑같이 겪을 문제인가?" → 예면 여기, 아니면 qa-config.

### KF-01 · <한 줄 제목>

- **발견 시점**: P<n> / run-<id> (해당되는 것)
- **킷 위치**: <어느 파일·어느 절이 문제인가> (예: phases/P2-artifacts.md §1.4)
- **무엇이 문제였나**: 지시가 불명확했나 / 코드만으로 부족했나 / 템플릿에 있어야 하는데 없었나 / 모순인가

- **일반성**: 다른 프로젝트에도 해당? (예/아니오 + 근거 한 줄)
- **상태**: open / 킷반영됨 / 기각(사유)
